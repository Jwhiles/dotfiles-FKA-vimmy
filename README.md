# VIMRC AND THEM PLUGINS

to install

```
cd ~
git clone git@github.com:Jwhiles/.vim.git
ln -s ~/.vim/vimrc ~/.vimrc
cd ~/.vim
git submodule init
git submodule update
```

## Extras
upgrade all plugins
`git submodule foreach git pull origin master`
